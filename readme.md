# はじめに
このリポジトリは、Udemy講座 [設計を理解しながらWebアプリを開発！【はじめてのWeb開発】【Flask】][1]の成果物に私(Souken521)なりの改良を施したものです。

# 目的の限定について
本リポジトリは、私のポートフォリオ目的の利用に限らせていただきます。

## 追加・改善した機能

- ログイン中のユーザーに紐づくメモのみ表示するように変更
- メモの登録者情報をDBに保存し、ユーザーごとのデータ管理に対応
- メモの作成日時・更新日時を表示
